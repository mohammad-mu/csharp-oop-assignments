## 📝 Assignment

Create a Student class with the following attributes:
* **Name**
* **ID**
* **GPA**

The class should also have a method **DisplayInfo()** that prints the student's details.

Then, create an object of **Student**, assign values to its attributes, and call **DisplayInfo()**.
